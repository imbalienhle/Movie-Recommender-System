# Movie-Recommender-System

The Movie Recommender System is a full stack  machine learning web application that recommends movies based on similarity and user preference built with Streamlit(Front-end), FastApi(Back-end), and Skikit-learn(Machine Learning, TF-IDF Vectorizer and Cosine Similarity).

# Features
- 💿Interactive user interface
- 🎬Recommends 10 Movies based on searched title
- 🖼Movie poster display
- ⚡Fast API backend integration
- 🛸Machine Learning recommendation engine
- 🔐Full stack integration

# Architechure
```
Frontend (Streamlit)
      |
      | User Requests
      v
Backend (FastAPI)
      |
      | Processes Recommendation
      v
Scikit-learn (TF-IDF + Cosine Similarity)
      |
      | Use Movie Dataset
      v
Movie Dataset (pandas/NumPy)
      |
      | Fetches Movie Posters
      v
TMDB API
```
